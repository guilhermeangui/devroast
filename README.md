# DevRoast

Paste your code. Get roasted.

DevRoast is an AI-powered code review app that gives your code a brutally honest score and roast. Submit any code snippet and watch it get rated, criticized, and ranked on the shame leaderboard.

Built during the **NLW (Next Level Week)** event by [Rocketseat](https://rocketseat.com.br), following the event classes and lessons.

## Features

- **Code Roasting** — paste any code snippet and get an AI-generated review with a score from 0 to 10
- **Roast Mode** — toggle maximum sarcasm for extra brutal reviews
- **Shame Leaderboard** — the worst code on the internet, ranked by score
- **Syntax Highlighting** — code is displayed with proper syntax highlighting
- **Diff View** — see suggested improvements as added/removed lines

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Scripts

| Command | Description |
|---|---|
| `pnpm dev` | Start dev server with Turbopack |
| `pnpm build` | Production build |
| `pnpm lint` | Run Biome linter |
| `pnpm format` | Auto-format with Biome |
